# BIFROST Browser Extension

Manifest V3 browser extension for clipping the current tab into the BIFROST PWA.

## Local Install

1. Open Chrome or Edge extensions.
2. Enable developer mode.
3. Choose "Load unpacked".
4. Select this folder: `extensions/browser`.

The popup reads the active tab title and URL, detects common chapter URL
patterns, and opens BIFROST with the Add Bookmark modal prefilled.
